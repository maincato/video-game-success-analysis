# Data Folder

The dataset is not included in this repository.

To run the notebook locally, place the CSV file in this folder and name it:

`video_game_sales_ratings.csv`

## Source
Download the dataset from the Kaggle page for **Video Game Sales with Ratings** and place the CSV file in this folder.

[Dataset link](https://www.kaggle.com/datasets/rush4ratio/video-game-sales-with-ratings/data)

## Note
The dataset is used locally for analysis and is not redistributed in this repository.
