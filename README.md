# What Makes a Video Game Successful?
### An Analysis of Genre, Platform, Ratings, and Sales

## Overview
This project explores the factors associated with video game success by analyzing genre, platform, critic and user ratings, and sales data.

The goal is to identify patterns that may help explain which types of games perform best and how review scores, platforms, and genres relate to commercial performance.

## Objectives
- Analyze video game sales across genres and platforms
- Compare critic scores and user scores
- Explore whether highly rated games tend to sell more
- Identify trends in game releases over time
- Summarize the main factors associated with video game success
